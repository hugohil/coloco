# Coloco Server

Coloco Sound System is a local network sound system.

It is still a work in progress.

## Requirements

- Coloco [remote](http://github.com/hugohil/coloco-remote)
- Coloco [player](http://github.com/hugohil/coloco-player)

## Installation

```bash
$ npm install
$ mkdir sounds
```

## Run

```bash
$ npm start
```
